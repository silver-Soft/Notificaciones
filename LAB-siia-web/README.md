# SIIA-5

Proyecto web desarrollado en **Angular 16**, orientado a la gestión y operación administrativa dentro del sistema SIIA.  
Este proyecto utiliza **Angular Material**, manejo de formularios reactivos, carga de archivos, compresión de imágenes, manejo de PDF, traducciones y otros módulos adicionales.

---

## 🛠️ Requerimientos del entorno

Antes de comenzar, asegúrate de tener instaladas las siguientes herramientas:

| Herramienta | Versión recomendada | Instalación / Verificación |
|------------|--------------------|----------------------------|
| **Node.js** | 18.x o superior | https://nodejs.org/ |
| **npm** | 9.x o superior | Incluido con Node.js |
| **Angular CLI** | 16.x | `npm install -g @angular/cli` |
| **Git** | Última versión | https://git-scm.com/ | (opcional si usas GitHub Desktop)

Verifica las versiones instaladas:

```bash
node -v
npm -v
ng version
